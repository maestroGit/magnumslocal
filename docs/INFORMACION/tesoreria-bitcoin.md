# ₿ Estrategia de Tesorería Bitcoin

## 🎯 Concepto Central

Transformar parte de los ingresos en Bitcoin (BTC) como activo de tesorería mediante:

- Pagos directos en BTC de wine lovers
- Conversión estratégica de euros a BTC

## 🔐 Estructura de Custodia Multifirma

### Configuración Técnica

| Elemento | Especificación |
|----------|----------------|
| Tipo de wallet | Multifirma 3 de 5 |
| Participantes | 5 bodegas fundadoras |
| Consenso mínimo | 3 firmas para mover fondos |
| Transparencia | Dirección BTC pública + informes |

### Modelo de Gobernanza

- **Representantes**: 1 delegado por bodega con clave y voto
- **Comité de firma**: Ejecuta transacciones con consenso
- **Asamblea digital**: Votaciones sobre uso de fondos
- **Auditoría pública**: Histórico visible en blockchain
