# 🍷 Guía de Prueba - Large Magnum Master Extension

## 🔧 **COMANDOS PARA PROBAR:**

### **Acceso Principal:**
1. Presiona `Ctrl+Shift+P` (Command Palette)
2. Escribe "Magnum Master"
3. Verás 4 comandos disponibles:

---

## 📋 **LISTA DE COMANDOS:**

### 1️⃣ **🍷 Ver Estado del Blockchain**
- **Comando**: `Magnum Master: Ver Estado del Blockchain`
- **Qué hace**: Abre dashboard visual con estado del sistema
- **Resultado esperado**: Página web con gráficos y estado

### 2️⃣ **📋 Información del Sistema**
- **Comando**: `Magnum Master: Información del Sistema`
- **Qué hace**: Muestra detalles técnicos en Output
- **Resultado esperado**: Panel Output con información detallada

### 3️⃣ **⛏️ Minar Nuevo Bloque**
- **Comando**: `Magnum Master: Minar Nuevo Bloque`
- **Qué hace**: Simulador interactivo de minado
- **Resultado esperado**: Input box para datos + simulación

### 4️⃣ **🔍 Ver Blockchain Completo**
- **Comando**: `Magnum Master: Ver Blockchain Completo`
- **Qué hace**: Explorador estilo terminal del blockchain
- **Resultado esperado**: Vista web con bloques estilo hacker
