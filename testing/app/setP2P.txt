1. Estos comandos configuran un nodo y lo inician:
set peers=ws://localhost:5001
set P2P_PORT=5002
set HTTP_PORT=3001
npm run dev

set peers=ws://localhost:5001: Establece la dirección del peer (otro nodo) al cual este nodo intentará conectarse. En este caso, ws://localhost:5001 indica que se intentará conectar a un nodo que escucha en el puerto 5001 en la misma máquina (localhost).
set P2P_PORT=5002: Configura el puerto en el que este nodo escuchará conexiones P2P. Aquí, el nodo escuchará en el puerto 5002.
set HTTP_PORT=3001: Configura el puerto en el que este nodo ofrecerá servicios HTTP. Aquí, el servicio HTTP estará disponible en el puerto 3001.
npm run dev: Ejecuta el comando para iniciar el nodo en modo de desarrollo. 

2. Iniciar el Segundo Nodo
Comandos:
sh
set peers=ws://localhost:5002
set P2P_PORT=5003
set HTTP_PORT=3002
npm run dev

Aquí estamos configurando y arrancando un segundo nodo con diferentes puertos:
set peers=ws://localhost:5002: Este nodo intentará conectarse a un nodo escuchando en ws://localhost:5002 (el puerto P2P del primer nodo).
set P2P_PORT=5003: Configura el puerto P2P de este segundo nodo a 5003.
set HTTP_PORT=3002: Configura el puerto HTTP de este segundo nodo a 3002.
npm run dev: Inicia el segundo nodo en modo de desarrollo.

3. Iniciar el Tercer Nodo
Comandos:
sh
set peers=ws://localhost:5003
set P2P_PORT=5004
set HTTP_PORT=3003
npm run dev
Finalmente, configuramos y arrancamos un tercer nodo:
set peers=ws://localhost:5003: Este nodo intentará conectarse a un nodo escuchando en ws://localhost:5003 (el puerto P2P del segundo nodo).
set P2P_PORT=5004: Configura el puerto P2P de este tercer nodo a 5004.
set HTTP_PORT=3003: Configura el puerto HTTP de este tercer nodo a 3003.
npm run dev: Inicia el tercer nodo en modo de desarrollo.
Resumen del Proceso
Primer Nodo: Configurado para conectarse a sí mismo (peer en localhost:5001), escuchar en P2P_PORT=5002 y ofrecer servicios HTTP en HTTP_PORT=3001.
Segundo Nodo: Se conecta al primer nodo (localhost:5002), escucha en P2P_PORT=5003 y ofrece servicios HTTP en HTTP_PORT=3002.
Tercer Nodo: Se conecta al segundo nodo (localhost:5003), escucha en P2P_PORT=5004 y ofrece servicios HTTP en HTTP_PORT=3003.
Cada nodo está configurado para conectarse al nodo anterior en la cadena, creando una red P2P local. Al ejecutar npm run dev, cada nodo se inicia con las configuraciones especificadas, lo que permite que se comuniquen entre ellos.

export P2P_PORT=5004
export HTTP_PORT=3003
